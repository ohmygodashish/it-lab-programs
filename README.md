# IT Lab Programs
This repository contains the laboratory programs implemented as part of the Information Technology minor degree coursework.

## Repository Structure
```
├── IT201n/          # Course: [Data Structures]
├── IT301n/          # Course: [Compiler Design]   
└── IT303/           # Course: [Computer Networks]
```

## How to Use
1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd it-lab-programs
   ```

2. Navigate to the desired course folder:
   ```bash
   cd IT301n
   ```

3. Compile any experiment using a C++ compiler:
   ```bash
   g++ -o exp1 EXP1.cpp
   ```

4. Run the executable:
   ```bash
   ./exp1
   ```

## Requirements
- C++ compiler (g++ recommended)
- Standard C++ libraries
- Command line/terminal environment

## Contributing
This repository is primarily for educational purposes. If you're a student contributing to this repository:
1. Follow the existing naming conventions (EXP1.cpp, EXP2.cpp, etc.)
2. Include proper comments explaining the implementation
3. Ensure code is well-formatted and readable
4. Add a brief description of the experiment at the top of each file
